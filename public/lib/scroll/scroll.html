scroll兼容IE8的自定义滚动条插件（无需实例化）

功能样式说明

插件版本：v2.0（初级测试版）

简要说明：本插件，可以用于比较简易交互的内容宽中优化模块的滚动条，替换浏览器自带滚动条样式修改不方便等问题。基本兼容于IE8浏览器。插件中能够提供基本的纵向、横向滚动条，支持修改滚动条样式，滚动条自动监控内容大小变化重绘滚动条等功能，并提供相应的回调函数。

使用说明

属性 说明 类型 默认值
skin 自定义class样式名称，会直接add到调用的元素上 string 空
conced 滚动条与内容间的距离，false时为覆盖在内容上方；有距离值时为内容的padding值； int 0
standBar 纵向滚动条的显示开关，控制是否显示纵向滚动条 boolean true
acrossBar 横向滚动条的显示开关，控制是否显示横向滚动条 boolean true
borderRadius 设置滚动条圆角尺寸 string 0px
scrollWidth 滚动条的宽度，纵向滚动条时为滚动条的宽度值，横向滚动条时为高度值 string 8px
bottomFixed 默认始终从底部显示，无论模块内大小改变，都默认滚动到底部，从底部显示起。此处只对纵向滚动条产生影响。 boolean false
rightFixed 默认始终从右部显示，无论模块内大小改变，都默认滚动到右部，从右部显示起。此处只对横向滚动条产生影响。 boolean false
autoResetRraw 是否开启自动监控内容大小变化，触发滚动条重绘功能。在固定的内容（或固定的滚动宽高），建议不开启，可节省浏览器资源。 boolean false
autoHideScroll 是否自动隐藏滚动条。本API设置为true的时候，鼠标hover入滚动内容时则显示滚动条，移除内容时则隐藏滚动条。false时，则始终显示滚动条。 boolean false
beforeFn 回调函数。在滚动条未挂载到HTML上是触发。 object funcion
afterFn 回调函数。在滚动条刚刚挂载到HTML上的时候（初始化后）触发。返回一个回调参数，该参数是开放出来的功能对象，带有五个函数：toScrollTop（可以手动触发滚动条滚动到顶部）；toScrollBottom（可以手动触发滚动条滚动到底部）；toScrollLeft（可以手动触发滚动条滚动到左部）；toScrollRight（可以手动触发滚动条滚动到右部）；restDrawScroll（可以手动触发滚动条重绘计算）。 object funcion
scrollFn 回调函数。在鼠标点击滚动条后滚动滚动条过程中（mousemove）触发，返回2个参数。第1个参数是具体滚动的方向：x/y（string类型）；第2个参数是当前内容的滚动距离。 object funcion
afterScrollFn 回调函数。在鼠标点击滚动条滚动到停止松开鼠标（mouseup）或者是滚轮滚动停下时触发，返回2个参数。第1个参数是具体滚动的方向：x/y（string类型）；第2个参数是当前内容的滚动距离。 object funcion
